# 🇮🇳 India Electricity Consumption Analysis

## Trend Identification & Anomaly Detection using Python, SQL and Looker Studio

---

## Project Overview

This project analyzes India's state-wise electricity consumption from 2013 to 2024 using Python, BigQuery SQL, and Looker Studio.

The objective is to identify consumption trends, seasonal patterns, and unusual demand spikes while building an end-to-end analytics pipeline similar to real-world business intelligence workflows.

---

## Objectives

- Clean and preprocess electricity consumption data
- Perform exploratory data analysis (EDA)
- Analyze electricity demand using SQL
- Detect unusual consumption patterns (anomalies)
- Build an interactive dashboard
- Generate business insights

---

## Dataset

Source:
Kaggle – India's Electricity Consumption Dataset

Duration:
2013–2024

Records:
3707 daily observations

Coverage:
Indian States and Union Territories

## Key Insights

- Maharashtra recorded the highest electricity consumption during the study period.
- Electricity demand shows a consistent increasing trend over the years.
- Summer months exhibit higher electricity demand compared to other seasons.
- Several states experienced significant demand spikes detected using rolling average analysis.
